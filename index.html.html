Smart Classroom Learning System

Prototype for Engaging Students with Mixed Intellectual Abilities

About the Prototype

This system is designed to improve classroom engagement for students with different intellectual levels. It uses smart teaching methods such as interactive quizzes, visual learning, and adaptive content.

Key Features

  • Interactive quizzes during class
  • Visual learning through videos and diagrams
  • Adaptive learning content for different skill levels
  • Instant feedback for students
  • Teacher dashboard to track student understanding
